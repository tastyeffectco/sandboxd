# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1][cc]** as its code
of conduct. Please read the full text at the link below — it defines the
standards of behavior we expect and the responsibilities of maintainers.

- Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## In short

We want participation in sandboxd to be a harassment-free experience for
everyone. Be respectful, assume good faith, welcome newcomers, and focus on
what is best for the community. Harassment, discrimination, personal attacks,
and other unacceptable behavior described in the Covenant are not tolerated —
in issues, pull requests, discussions, or any other project space.

## Enforcement

Report unacceptable behavior privately to the maintainers via GitHub — open a
[private vulnerability report](https://github.com/tastyeffectco/sandboxd/security/advisories/new)
(the same private channel; it works for conduct reports too) or contact
[**@tastyeffectco**](https://github.com/tastyeffectco). All reports are reviewed
and handled confidentially. Maintainers who do not follow or enforce
this code in good faith may face repercussions decided by other project leaders.

Enforcement follows the Contributor Covenant's
[Enforcement Guidelines][guidelines].

[cc]: https://www.contributor-covenant.org
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
